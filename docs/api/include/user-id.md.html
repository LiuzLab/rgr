User by ID

Search for public users using a userId.

  • URL

    /api/users/{userId}

  • Method:

    GET

  • URL Params

    Required:
    None

    Optional:
    auth=[alphanumeric] : Authentication token.

  • Success Response:

    Returns a user information for {userId}.
